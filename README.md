# C Practice Programs

This repository contains a collection of C programs written while learning programming fundamentals, mainly inspired by CS50 and basic problem-solving exercises.

---

## Files and Descriptions

### cash.c
Implements the CS50 Cash problem.
- Takes an amount of change owed as input.
- Calculates the minimum number of coins required using a greedy algorithm.
- Coins used: 25, 10, 5, and 1.

---

### left_triangle.c
- Prints a left-aligned triangle pattern using #.
- Demonstrates nested loops.

---

### right_triangle.c
- Prints a right-aligned triangle pattern.
- Focuses on spacing and alignment logic.

---

### diamond.c
- Prints a diamond-shaped pattern.
- Combines upper and lower triangle logic.

---

### reverse_no.c
- Reverses a given number.
- Example: 123 → 321

---

### swap_no.c
- Swaps two numbers.
- Can be implemented with or without a temporary variable.

---

### palindrome_check.c
- Checks whether a number is a palindrome.
- Example: 121 → palindrome, 123 → not palindrome

---

### factorial.c
- Calculates the factorial of a number.
- Example: 5! = 120

---

### fibonacci.c
- Prints Fibonacci sequence up to n terms.

---

### triangle.c
- General triangle pattern program.

---

## Concepts Covered
- Loops (for, while)
- Conditionals (if-else)
- Functions
- Pattern printing
- Number manipulation
- Basic algorithms
